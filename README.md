# H5 移动端开发脚手架

这是一个简单、实用、基础的 H5 移动端开发脚手架。

## 克隆本仓库

```bash
git clone https://github.com/cntanglijun/h5-mobile-scaffold.git <your-dir-name>
```

## 安装依赖模块

```bash
npm install
```

## 启动服务器

```bash
npm start
```

## 打包发布

```bash
npm run build
```
